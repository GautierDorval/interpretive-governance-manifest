# CCL (Causal Context Layer)

Status: proposed  
Normative language: MUST / SHOULD / MAY

CCL is an Interpretive Governance extension for declaring the context of necessity of a content surface, claim, route, or service bridge.

It preserves the relation between:

```text
trigger situation -> problem or risk -> latent need -> canonical surface -> intended consequence -> prohibited derivations
```

CCL does not measure semantic proximity and does not guarantee outcomes.
