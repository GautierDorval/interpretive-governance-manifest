# Semantic-boundary

Status: proposed  
Normative language: MUST / SHOULD / MAY

Semantic-boundary declares non-equivalence, false-neighbor risks, and proximity-risk boundaries between concepts, claims, entities, or surfaces.

It prevents a common failure mode: proximity becomes fusion.
