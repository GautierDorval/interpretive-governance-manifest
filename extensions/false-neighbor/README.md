# False-neighbor extension profile

Status: proposed  
Depends on: `extensions/semantic-boundary/semantic-boundary.md`

This profile defines a portable shape for interpretive false-neighbor declarations.

An interpretive false neighbor is a concept, claim, entity, or surface that is close enough to another element to be plausibly fused by a machine system, while remaining non-equivalent under the governing doctrine.
