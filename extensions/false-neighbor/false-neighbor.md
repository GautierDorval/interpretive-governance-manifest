# False-neighbor extension profile

Status: proposed  
Normative language: MUST / SHOULD / MAY

## Purpose

False-neighbor declarations prevent plausible fusion between close concepts.

## Minimal record

A false-neighbor record MUST declare:

- source term or surface;
- neighbor term or surface;
- relation type;
- confusion risk;
- prohibited fusion;
- required behavior.

## Measurement boundary

A false-neighbor declaration is not behavioral evidence.

A testset may propose prompts and expected boundaries, but evidence requires a separate evaluation run with model declarations, judge criteria, timestamps, outputs, and results.
