# Extensions

This directory contains additive modules for the Interpretive Governance standard.

Extensions must not redefine the baseline ontology, statement typing, interpretive/constraintive separation, or authority governance boundary.

## Current extension families

- `ctic/` - Cross-layer transactional integrity.
- `ccl/` - Causal Context Layer, proposed.
- `semantic-boundary/` - Non-equivalence and proximity-risk boundaries, proposed.
- `false-neighbor/` - False-neighbor declaration profile, proposed.
- `maximalist-governance-package/` - Governance package inventory profile, proposed.
- `measurement/` - Experimental report shapes for future behavioral evaluation.

## Maturity statuses

- `draft`: under active definition.
- `proposed`: coherent enough for early implementations, not yet stabilized.
- `experimental`: measurement, run, or instrumentation shape without stable guarantees.
- `stable`: stabilized after implementation feedback and validation.
- `deprecated`: retained for compatibility, no longer recommended.
